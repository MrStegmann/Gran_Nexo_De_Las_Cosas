
# Convertir archivos md a componentes react

## src\features\sheets
Escanea los archivos md de src\assets\documents\sheet y crea un componente tsx por cada uno con html semántico.

- Estos componentes deben estar en src\features\sheets\components.
- Crear una nueva carpeta para src\features\sheets\components\SheetsFeature.tsx que sea Screen y mueve el archivo a esa carpeta.
- Añade todos los nuevos componentes a sheets, importando debidamente los componentes.
- Comprueba que SheetsFeature esté bien importado.

## src\features\inventory
Escanea los archivos md de src\assets\documents\inventory y crea un componente tsx por cada uno con html semántico.

- Asegurate de que las imagenes en src\assets\documents\inventory\img esten correctamente importadas en los componentes.
- Crear una nueva carpeta para src\features\inventory\components\InventoryFeature.tsx que sea Screen y mueve el archivo a esa carpeta.
- Añade todos los nuevos componentes a sheets, importando debidamente los componentes.
- Comprueba que SheetsFeature esté bien importado.

## Acceptance Criteria:
- Los componentes deben usar html semántico.
- Debe usarse tailwindcss para los estilos, siguiendo el mismo estilo general del proyecto.
